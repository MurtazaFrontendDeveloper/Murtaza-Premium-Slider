# Murtaza-Premium-Slider ✨

A modern, responsive and **premium image slider** built with vanilla JavaScript, HTML, and CSS. Designed with a dark + cyan aesthetic for a smooth and elegant UI experience.

---

## 🚀 Features

- Smooth image sliding animation
- Auto-play functionality (3s interval)
- Manual navigation (Next / Prev buttons)
- Fully responsive design (mobile, tablet, desktop)
- Clean glassy premium UI design
- Dark + cyan modern theme
- Auto-reset timer on interaction

---

## 🎨 Tech Stack

- HTML5
- CSS3 (Flexbox + Gradients)
- JavaScript (Vanilla)

---

## 📱 Responsive Design

This slider is fully responsive and adapts to all screen sizes:
- Mobile 📱
- Tablet 📟
- Desktop 💻

Images scale smoothly and layout stays centered on all devices.

---

## 📸 Preview

A sleek slider with cinematic images and a premium dark-glass design.

---

## ⚙️ How it works

- Images are shifted using `translateX`
- Auto-slide runs using `setTimeout`
- Manual click resets the timer for smooth control
- Responsive width is calculated dynamically
